Area
===============================================================
Area is a Wordpress theme especially developed for dito.areato.org website.

====================================================
Requirements
====================================================
Wordpress 3.x +.

====================================================
Usage
====================================================
Simply add the Area folder to the theme folder and activate the plugin from the admin appereance section.
